Yardım Noktası (Postman Manuel Test)

          

            Base URL: http://localhost:8080
            

            1) Register → POST /auth/register body:
            {"username":"admin","password":"123456"}
            

            2) Login → POST /auth/login body:
            {"username":"admin","password":"123456"}
            

            3) Dönen tokenı Postman'de tüm korumalı çağrılarda Header olarak ekle:
            Authorization: Bearer <TOKEN>
            

            4) Test et:
            GET /profile, GET /users, GET /conversations
            

            5) Konuşma oluştur:
            POST /conversations body:
            {"participantUsername":"alice2"}
            

            6) Mesaj gönder:
            POST /conversations/{id}/messages body:
            {"text":"merhaba"}
            

            7) Mesajları getir:
            GET /conversations/{id}/messages
